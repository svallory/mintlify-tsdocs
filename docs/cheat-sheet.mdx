## Installation

```bash
bun install -g mint-tsdocs   # Global install
npx mint-tsdocs              # No install needed
```

---

## Commands Quick Reference

| Command | Shorthand | Description |
|---------|-----------|-------------|
| `mint-tsdocs` | - | Defaults to `generate` |
| `mint-tsdocs init [DIR]` | - | Initialize project configuration |
| `mint-tsdocs generate [DIR]` | `mint-tsdocs [DIR]` | Generate documentation |
| `mint-tsdocs customize -t DIR` | - | Copy templates for customization |
| `mint-tsdocs show [config\|stats]` | - | Display configuration or cache stats |
| `mint-tsdocs help` | `-h`, `--help` | Show help |
| `mint-tsdocs version` | `-v`, `--version` | Show version |

## Global Flags

| Flag | Short | Description |
|------|-------|-------------|
| `--verbose` | `-v` | Verbose output (info level) |
| `--debug` | - | Debug output (all levels) |
| `--quiet` | `-q` | Errors only |
| `--help` | `-h` | Show help |

## init Command

**Syntax:**
```bash
mint-tsdocs init [PROJECT_DIR] [OPTIONS]
```

**Options:**

| Option | Short | Description |
|--------|-------|-------------|
| `--yes` | `-y` | Skip prompts (use defaults) |
| `--skip-mintlify` | - | Don't initialize Mintlify |
| `--project-dir PATH` | - | Project directory |

**Examples:**
```bash
mint-tsdocs init                          # Interactive setup
mint-tsdocs init --yes                    # Quick setup
mint-tsdocs init ./packages/core          # Monorepo package
mint-tsdocs init --skip-mintlify          # Skip Mintlify setup
```

---

## generate Command

**Syntax:**
```bash
mint-tsdocs generate [PROJECT_DIR] [OPTIONS]
mint-tsdocs [PROJECT_DIR]                 # Shorthand
```

**Options:**

| Option | Description |
|--------|-------------|
| `--skip-extractor` | Skip api-extractor (use existing .api.json) |
| `--project-dir PATH` | Project directory |

**Examples:**
```bash
mint-tsdocs                               # Current project
mint-tsdocs ./packages/core               # Specific package
mint-tsdocs generate --verbose            # Verbose output
mint-tsdocs generate --skip-extractor     # Skip api-extractor
mint-tsdocs generate --debug              # Debug mode
```

---

## customize Command

**Syntax:**
```bash
mint-tsdocs customize -t TEMPLATE_DIR [OPTIONS]
```

**Options:**

| Option | Short | Description |
|--------|-------|-------------|
| `--template-dir PATH` | `-t` | Template directory (required) |
| `--force` | - | Overwrite existing files |

**Examples:**
```bash
mint-tsdocs customize -t ./templates      # Copy templates
mint-tsdocs customize -t ./templates --force  # Overwrite
```

## show Command

**Syntax:**
```bash
mint-tsdocs show [config|stats]
mint-tsdocs show --target [config|stats]
```

**Targets:**

| Target | Default | Description |
|--------|---------|-------------|
| `config` | ✓ | Show configuration |
| `stats` | | Show cache statistics |

**Examples:**
```bash
mint-tsdocs show                          # Show config (default)
mint-tsdocs show config                   # Show config
mint-tsdocs show stats                    # Show cache stats
```

## Configuration File

**Location:** `mint-tsdocs.config.json` (project root)

**Minimal:**
```json
{
  "entryPoint": "./lib/index.d.ts",
  "outputFolder": "./docs/reference"
}
```

**Common Options:**

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `entryPoint` | string | - | TypeScript entry point (.d.ts) |
| `outputFolder` | string | `./docs/reference` | MDX output directory |
| `docsJson` | string | `./docs/docs.json` | Mintlify docs.json path |
| `tabName` | string | `API Reference` | Navigation tab name |
| `groupName` | string | `API` | Navigation group name |
| `convertReadme` | boolean | `false` | Convert README to MDX |
| `readmeTitle` | string | `README` | README page title |

**Templates:**
```json
{
  "templates": {
    "userTemplateDir": "./templates",
    "cache": true,
    "strict": true
  }
}
```

**API Extractor:**
```json
{
  "apiExtractor": {
    "configPath": "./api-extractor.json",  // Optional custom config
    "bundledPackages": [],
    "compiler": {
      "tsconfigFilePath": "./tsconfig.json"
    },
    "docModel": {
      "projectFolderUrl": "https://github.com/user/repo"
    }
  }
}
```

## Common Workflows

### Initial Setup
```bash
mint-tsdocs init                          # 1. Initialize
bun run build                             # 2. Build TypeScript
mint-tsdocs                               # 3. Generate docs
```

### Development
```bash
# Make changes → rebuild → regenerate
vim src/index.ts
bun run build
mint-tsdocs
```

### Monorepo Setup
```bash
# Initialize all packages
for pkg in packages/*; do
  mint-tsdocs init "$pkg" --yes
done

# Generate all docs
for pkg in packages/*; do
  mint-tsdocs "$pkg"
done
```

### Template Customization
```bash
mint-tsdocs customize -t ./templates      # 1. Copy templates
vim ./templates/class.liquid              # 2. Edit templates
# Update config: "templates": { "userTemplateDir": "./templates" }
mint-tsdocs                               # 3. Regenerate
```

## package.json Scripts

**Recommended Scripts:**
```json
{
  "scripts": {
    "docs:init": "mint-tsdocs init",
    "docs:generate": "mint-tsdocs generate",
    "docs": "mint-tsdocs",
    "docs:dev": "mintlify dev"
  }
}
```

**Usage:**
```bash
bun run docs:init                         # Initialize
bun run docs                              # Generate
bun run docs:dev                          # Preview
```


```bash
mint-tsdocs generate --skip-extractor
```

## Environment Variables

### DEBUG Variable

**Levels:**
```bash
# Errors only
DEBUG=mint-tsdocs:*:error mint-tsdocs

# Warnings and errors (default without DEBUG)
mint-tsdocs

# Info, warnings, errors (same as --verbose)
DEBUG=mint-tsdocs:*:info mint-tsdocs

# All output (same as --debug)
DEBUG=mint-tsdocs:* mint-tsdocs
```

**Specific Namespaces:**
```bash
DEBUG=mint-tsdocs:cli:* mint-tsdocs       # CLI only
DEBUG=mint-tsdocs:cache:* mint-tsdocs     # Cache only
DEBUG=mint-tsdocs:cli:*,mint-tsdocs:cache:* mint-tsdocs  # Multiple
```

## File Structure

```
project-root/
├── mint-tsdocs.config.json          # Main config
├── package.json
├── src/
│   └── index.ts                     # Source code
├── lib/
│   └── index.d.ts                   # Built declarations (entry point)
├── templates/                       # Custom templates (optional)
│   ├── layout.liquid
│   ├── class.liquid
│   └── interface.liquid
└── docs/
    ├── docs.json                    # Mintlify navigation
    ├── reference/                   # Generated MDX files
    │   ├── index.mdx
    │   ├── myclass.mdx
    │   └── myinterface.mdx
    └── .tsdocs/                     # Cache (gitignored)
        ├── api-extractor.json       # Generated config
        ├── tsdoc.json               # Generated config
        └── *.api.json               # API model files
```

## Troubleshooting Quick Fix

| Error | Solution |
|-------|----------|
| Config not found | `mint-tsdocs init` |
| `declaration: true` not set | Select fix when prompted or update tsconfig.json |
| API Extractor errors | Check TypeScript build: `ls lib/*.d.ts` |
| Template errors | `mint-tsdocs generate --debug` |
| Cache issues | `rm -rf docs/.tsdocs && mint-tsdocs` |

## Quick Tips

- **Default action**: `mint-tsdocs` = `mint-tsdocs generate`
- **Monorepo**: Use positional args: `mint-tsdocs ./packages/core`
- **No install**: Use `npx mint-tsdocs`
- **Debug mode**: Add `--debug` to any command
- **Quiet mode**: Add `--quiet` for CI/CD
- **Custom templates**: Run `customize` then edit `.liquid` files
- **Cache location**: `docs/.tsdocs/` (auto-created, gitignored)
- **Config schema**: Add `$schema` for IDE autocomplete
