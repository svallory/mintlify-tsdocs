## Prerequisites

  <Card title="Node.js" icon="node-js">
    Node.js 18 or higher
  </Card>

    A TypeScript library or package

## Step 1: Install Dependencies

<CodeGroup>

```bash npm
npm install -D @microsoft/api-extractor mintlify-tsdocs
```

```bash bun
bun add -D @microsoft/api-extractor mintlify-tsdocs
```

```bash yarn
yarn add -D @microsoft/api-extractor mintlify-tsdocs
```

</CodeGroup>

## Step 2: Configure API Extractor

Create `api-extractor.json` in your project root:

```json api-extractor.json
{
  "$schema": "https://developer.microsoft.com/json-schemas/api-extractor/v7/api-extractor.schema.json",
  "mainEntryPointFilePath": "<projectFolder>/dist/index.d.ts",
  "apiReport": {
    "enabled": false
  },
  "docModel": {
    "enabled": true,
    "apiJsonFilePath": "<projectFolder>/temp/<unscopedPackageName>.api.json"
  },
  "dtsRollup": {
    "enabled": false
  }
}
```

<Tip>
  The `<projectFolder>` and `<unscopedPackageName>` tokens are replaced automatically by API Extractor based on your package.json.
</Tip>

## Step 3: Build and Extract

```bash
# Build your TypeScript project
npm run build

# Extract API documentation
api-extractor run --local
```

You should now have a `.api.json` file in your `temp/` directory.

## Step 4: Generate Documentation

```bash
mintlify-tsdocs markdown \
  -i ./temp \
  -o ./docs/api \
  --docs-json ./docs/docs.json \
  --tab-name "API Reference" \
  --group "Core API"
```

<ParamField path="i" type="string" required>
  Input folder containing `*.api.json` files
</ParamField>

<ParamField path="o" type="string" required>
  Output folder for generated MDX files (**contents will be deleted!**)
</ParamField>

<ParamField path="docs-json" type="string">
  Path to Mintlify `docs.json` file to update with navigation
</ParamField>

<ParamField path="tab-name" type="string">
  Tab name in Mintlify navigation (default: "API Reference")
</ParamField>

<ParamField path="group" type="string">
  Group name within the tab for organizing pages
</ParamField>

## Step 5: Preview Your Docs

```bash
# Install Mintlify CLI if you haven't already
npm install -g mintlify

# Start the local preview
cd docs
mintlify dev
```

Your documentation should now be available at `http://localhost:3000`!

## What You'll See

After generation, your `docs/api/` folder will contain:

<Steps>
  <Step title="Index Page">
    `index.mdx` - Package overview with description and exports list
  </Step>
  <Step title="API Item Pages">
    Individual MDX files for each exported class, interface, function, type, etc.
  </Step>
  <Step title="Updated Navigation">
    Your `docs.json` file will be updated with navigation entries for all generated pages
  </Step>
</Steps>

## Example Output

For a simple interface like this:

```typescript
/**
 * Configuration for database connection
 */
export interface DatabaseConfig {
  /**
   * Database host address
   */
  host: string;

  /**
   * Database port number
   */
  port: number;

  /**
   * Connection options
   */
  options?: {
    ssl: boolean;
    timeout: number;
  };
}
```

You'll get an interactive MDX page with:

<ParamField name="host" type="string" required>
  Database host address
</ParamField>

<ParamField name="port" type="number" required>
  Database port number
</ParamField>

<ParamField name="options" type="object">
  Connection options

  <ParamField name="ssl" type="boolean" required>
  </ParamField>

  <ParamField name="timeout" type="number" required>
  </ParamField>
</ParamField>

<Check>
  **You're all set!** Your API documentation is now live and interactive.
</Check>
