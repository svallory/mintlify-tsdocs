## 1. Initialize

Run this in your TypeScript project:

```bash
npx mint-tsdocs
```

The tool will:
- Auto-detect your TypeScript entry point
- Find your Mintlify docs folder
- Ask a few configuration questions
- Set everything up

<Tip>
  Use `--yes` to accept all defaults: `npx mint-tsdocs --yes`
</Tip>

## 2. Generate

```bash
npx mint-tsdocs generate
```

Done! Your API documentation is now in your docs folder.

## 3. Preview

```bash
cd docs
mintlify dev
```

Your docs are live at `http://localhost:3000`.

<Note>
  Don't have Mintlify installed? Run `npm install -g mintlify` first.
</Note>
