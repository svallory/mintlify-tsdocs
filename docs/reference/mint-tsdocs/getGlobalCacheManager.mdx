## Summary



## Parameters



### options


**Type**: <Link kind="ref" target="mint-tsdocs.CacheManagerOptions">`CacheManagerOptions`</Link>

## Returns



**Type**: <Link kind="ref" target="mint-tsdocs.CacheManager">`CacheManager`</Link>



Get the global cache manager instance




## Remarks

  The global cache manager is a singleton. Options can only be provided on the first call. If you need to reconfigure, call [object Object] first, or create a new instance with <code>new CacheManager(options)</code> instead.
