# writeTentative

## Summary

Writes `before` and `after` messages if and only if `mayWrite` writes anything.  If `mayWrite` writes &quot;CONTENT&quot;, this method will write &quot;CONTENT&quot;. If `mayWrite` writes nothing, this method will write nothing. 
## Signature

```typescript
writeTentative(before: string, after: string, mayWrite: () => void): void;
```

**Type**:`string`



### after
**Type**:`string`

**Type**:`() => void`
