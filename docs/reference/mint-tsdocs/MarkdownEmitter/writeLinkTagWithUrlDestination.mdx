# writeLinkTagWithUrlDestination

## Summary

Writes a link tag that references an external URL  
## Signature

```typescript
protected writeLinkTagWithUrlDestination(docLinkTag: DocLinkTag, context: IMarkdownEmitterContext): void;
```

## Parameters

### docLinkTag
**Type**:`DocLinkTag`

The link tag node with a URL destination

### context
**Type**:<Link kind="ref" target="mint-tsdocs.IMarkdownEmitterContext">`IMarkdownEmitterContext`</Link>

Emission context

## Returns

**Type**:`void`

Writes a link tag that references an external URL

## Remarks

  Handles external URL links like <code>{@link https://example.com}</code> or <code>{@link https://example.com | Link Text}</code>.  The link text is normalized (whitespace collapsed) and escaped for safe markdown output. The URL is used as-is.  Output format: <code>[Link Text](URL)</code>  Subclasses can override this to customize URL link rendering.
