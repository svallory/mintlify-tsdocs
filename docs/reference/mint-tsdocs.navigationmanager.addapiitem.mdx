# addApiItem

## Summary

Add an API item to navigation with automatic categorization

## Signature

```typescript
addApiItem(apiItem: any, filename: string, parentFilename?: string): void;
```

## Parameters

### apiItem
**Type**: `any`

The API item to add

### filename
**Type**: `string`

The filename of the generated page

### parentFilename
**Type**: `string`
**Optional**: Yes

Optional parent filename for member items

## Returns

**Type**: `void`

Add an API item to navigation with automatic categorization
