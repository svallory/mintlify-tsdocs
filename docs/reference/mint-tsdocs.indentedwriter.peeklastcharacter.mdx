# peekLastCharacter

## Summary

Returns the last character that was written, or an empty string if no characters have been written yet.

## Signature

```typescript
peekLastCharacter(): string;
```

## Parameters

*This method takes no parameters.*

## Returns

**Type**: `string`

Returns the last character that was written, or an empty string if no characters have been written yet.
