# sanitizeJsxAttribute

## Summary

Sanitizes text for safe use in JSX attributes. Prevents JSX injection and ensures proper escaping.


## Parameters

### text
**Type**: `string`

The text to sanitize

### attributeName
**Type**: `string`
**Optional**: Yes

The name of the JSX attribute (optional, for context-specific handling)

## Returns

**Type**: `string`

Sanitizes text for safe use in JSX attributes. Prevents JSX injection and ensures proper escaping.
