# increaseIndent

## Summary

Increases the indentation. Normally the indentation is two spaces, however an arbitrary prefix can optional be specified. (For example, the prefix could be &quot;// &quot; to indent and comment simultaneously.) Each call to IndentedWriter.increaseIndent() must be followed by a corresponding call to IndentedWriter.decreaseIndent().

## Signature

```typescript
increaseIndent(indentPrefix?: string): void;
```

## Parameters

### indentPrefix
**Type**: `string`
**Optional**: Yes
