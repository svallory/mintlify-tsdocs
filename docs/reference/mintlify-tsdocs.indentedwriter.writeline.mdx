# writeLine











## Summary

A shorthand for writing an optional message, followed by a newline. Indentation is applied following the semantics of IndentedWriter.write().





## Signature

```typescript
writeLine(message?: string): void;
```








## Parameters



### message


**Type**: string



**Optional**: Yes
